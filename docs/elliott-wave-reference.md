# Elliott Wave Principle — Complete Reference for Building an Analyzer

Source: "Elliott Wave Principle: Key to Market Behavior" by Frost & Prechter

## 1. FUNDAMENTAL STRUCTURE

### Basic Pattern
- Progress = 5 waves (motive); Corrections = 3 waves (corrective)
- Complete cycle = 8 waves (5 motive + 3 corrective)
- Subdivision: 2 → 8 → 34 → 144 waves (Fibonacci sequence)
- **Fractal**: all waves both HAVE and ARE component waves

### Two Modes
- **Motive**: 5-wave structure, moves WITH the trend of one larger degree
- **Corrective**: 3-wave structure (or variation), moves AGAINST the trend
- Mode is determined by RELATIVE direction, not absolute direction

### Wave Degrees (largest to smallest)
Grand Supercycle → Supercycle → Cycle → Primary → Intermediate → Minor → Minute → Minuette → Subminuette

### Labeling Convention
| Degree | Motive | Corrective |
|---|---|---|
| Supercycle | (I)(II)(III)(IV)(V) | (A)(B)(C) |
| Cycle | I II III IV V | A B C |
| Primary | [1][2][3][4][5] | [A][B][C] |
| Intermediate | (1)(2)(3)(4)(5) | (a)(b)(c) |
| Minor | 1 2 3 4 5 | a b c |

### Wave Function
- **Actionary labels**: 1, 3, 5, A, C, E, W, Y, Z
- **Reactionary labels**: 2, 4, B, D, X
- All reactionary waves develop in corrective mode
- Most actionary waves develop in motive mode (exceptions: ending diagonals, flats, triangles, combinations)

## 2. INVIOLABLE RULES (must NEVER be broken)

### R1: Wave 2 never retraces more than 100% of Wave 1
- Uptrend: Wave 2 low must stay ABOVE Wave 1 start
- Downtrend: Wave 2 high must stay BELOW Wave 1 start

### R2: Wave 3 is never the shortest of the three motive waves (1, 3, 5)
- Measure in price movement (percentage for large degrees, arithmetic for small)
- Most important for wave counting: if apparent wave 3 is too short, relabel as extended wave (3) in the making

### R3: Wave 4 does not enter the price territory of Wave 1 (no overlap)
- **Exception**: diagonal triangles (where overlap IS expected)
- **Exception**: leveraged/futures markets may show brief intraday violations

### R4: Corrections are never fives
- An initial 5-wave move against the larger trend is NEVER the end of a correction, only part of it

### R5: Motive waves always subdivide into 5 waves
- Actionary subwaves (1, 3, 5) of an impulse are themselves motive
- Wave 3 is SPECIFICALLY always an impulse (never a diagonal)

## 3. MOTIVE WAVE PATTERNS

### 3.1 Impulse (most common motive wave)
- Internal structure: **5-3-5-3-5**
- All three rules (R1, R2, R3) apply strictly
- Waves 1, 3, 5 = motive (subdivide into 5)
- Waves 2, 4 = corrective (subdivide into 3 or variation)

### 3.2 Extension
- One of the three actionary subwaves (1, 3, or 5) is elongated with exaggerated subdivisions
- Extensions occur in **only ONE** of the three actionary subwaves
- **Wave 3 extensions are most common** in stock/crypto markets
- If waves 1 and 3 are roughly equal → expect extended wave 5
- If wave 3 is extended → wave 5 should be simple, resembling wave 1
- Extensions within extensions possible (e.g., 3rd of extended 3rd)
- Extended wave produces 9-wave count of similar size instead of normal 5
- **Critical labeling rule**: When apparent wave 3 is too short or wave 4 overlaps wave 1, always check if it should be relabeled as an extended wave (3) in the making (Figure 1-8 — "single most useful guide")

### 3.3 Truncation (Truncated Fifth)
- Wave 5 fails to move beyond the end of Wave 3
- Must verify wave 5 contains its required 5 subwaves
- Usually occurs after an **extremely strong Wave 3**
- Signals **dramatic reversal** ahead

### 3.4 Ending Diagonal (wedge at termination points)
- Position: wave 5 (or wave C of A-B-C, or final C in double/triple threes)
- Internal structure: **3-3-3-3-3** (all subwaves are threes)
- Wave 4 overlaps wave 1 (exception to impulse overlap rule)
- Boundary lines converge (wedge shape)
- Wave 3 is still never the shortest
- Often ends with "throw-over" (brief break of 1-3 trendline)
- **Rising diagonal = bearish** → sharp decline follows
- **Falling diagonal = bullish** → sharp advance follows
- Indicates **exhaustion** of the larger movement
- Volume diminishes as pattern progresses; spike at throw-over

### 3.5 Leading Diagonal
- Position: wave 1 (or wave A of zigzag)
- Internal structure: **5-3-5-3-5** (differs from ending diagonal!)
- Wave 4 overlaps wave 1
- Boundary lines converge (wedge shape)
- Key identifier: **decided slowing of price change in 5th subwave relative to 3rd**
- Communicates "continuation" (vs ending diagonal's "termination")
- Don't confuse with series of first and second waves

## 4. CORRECTIVE WAVE PATTERNS

### 4.1 Zigzag (5-3-5) — Sharp correction
- Angles steeply against the larger trend
- Wave A = 5 waves, Wave B = 3 waves, Wave C = 5 waves
- Top of wave B is **noticeably lower** than start of wave A (bull market)
- Wave 2 frequently sports zigzags; wave 4 rarely does
- **Double zigzag** (W-X-Y): each W and Y is a zigzag, X connects (usually a zigzag)
- **Triple zigzag** (W-X-Y-X-Z): rare, each actionary component is a zigzag

### 4.2 Flat (3-3-5) — Sideways correction
- Less retracement than zigzags; participate in periods with strong underlying trend
- Wave A = 3 waves, Wave B = 3 waves, Wave C = 5 waves
- Three types:
  - **Regular flat**: B ends near start of A; C ends slightly past end of A
  - **Expanded flat** (most common!): B exceeds start of A; C ends well beyond end of A
  - **Running flat** (rare, strong trends only): B exceeds start of A; C falls short of end of A
- Wave 4 frequently sports flats; usually precede/follow extensions
- **Critical check**: If supposed B wave subdivides into 5 waves (not 3), it's likely wave 1 of next impulse, NOT a B wave
- Running flats: require internal subdivisions to strictly adhere to rules; label with extreme caution

### 4.3 Triangle (3-3-3-3-3) — Balance of forces
- Five overlapping waves labeled a-b-c-d-e
- Each subwave subdivides into 3 (usually zigzags, but c can be more complex)
- Delineate by connecting termination points of a-c and b-d
- Wave e can undershoot or overshoot the a-c line (happens more often than not)
- **Contracting types**: symmetrical, ascending, descending
- **Expanding type**: reverse symmetrical
- **Running triangle**: wave b exceeds start of wave a (extremely common)
- **Position**: almost always wave 4, wave B, or final wave X in a combination
- Rarely in wave 2 position
- **Thrust after triangle**: wave 5 following a triangle is swift, traveling ~distance of widest part of triangle
- In powerful markets: no thrust, but prolonged 5th wave instead
- Subwave c can be more complex (flat or multiple zigzag); rarely, subwave e is itself a triangle (9-wave total)
- Apex of contracting triangle often coincides with a turning point in time
- Associated with decreasing volume and volatility

### 4.4 Corrective Combinations
- **Double three** (W-X-Y): combination of simpler corrective patterns
- **Triple three** (W-X-Y-X-Z): same, with three components
- X waves are reactionary (usually zigzags)
- Rules for combinations:
  - A triangle is **only allowed as the final component**
  - **Only one zigzag** allowed per combination
  - **Never more than one triangle** per combination
  - Horizontal in character; extend the duration of correction
  - Component patterns commonly alternate in form (e.g., flat followed by triangle)

### Actionary Corrective Waves (corrective mode despite actionary direction)
- Waves 1, 3, 5 in an ending diagonal
- Wave A in a flat correction
- Waves A, C, E in a triangle
- Waves W, Y in double corrections
- Wave Z in triple corrections

### Wave Count Heuristic
- Motive: 5, 9, 13, 17 waves (few overlaps)
- Corrective: 3, 7, 11, 15 waves (numerous overlaps)
- If internal count is unclear, count waves to determine motive vs corrective

## 5. GUIDELINES (tendencies, not absolute rules)

### 5.1 Alternation
- **Within impulses**: If wave 2 is a sharp correction (zigzag), expect wave 4 to be sideways (flat/triangle), and vice versa
- Sharp corrections: never include a new price extreme beyond the orthodox end of the preceding impulse
- Sideways corrections: usually include a new price extreme beyond the orthodox end
- **Within corrections**: If wave A is a flat → expect wave B as zigzag, and vice versa
- Diagonal triangles do NOT display alternation between subwaves 2 and 4 (both usually zigzags)
- Extensions are an expression of alternation (motive waves alternate their lengths)

### 5.2 Depth of Corrective Waves
- Corrections (especially 4th waves) tend to retrace into the price territory of the **previous 4th wave of one lesser degree**
- Most commonly near the level of its terminus
- If wave 1 extends → correction after wave 5 typically retraces to the **bottom of wave 2** of lesser degree
- Flat/triangle corrections may barely fail to reach the 4th wave area
- Zigzags may cut deeper into 2nd wave area of lesser degree (almost exclusively when they are themselves 2nd waves)

### 5.3 Behavior After Fifth Wave Extensions
- Ensuing correction will be **sharp** and find support at the level of **wave 2 of the extension**
- Fifth wave extensions are typically followed by **swift retracements**
- This is the "most important empirically derived rule" — precision is remarkable
- Does NOT apply separately to 5th wave extensions of 5th wave extensions

### 5.4 Wave Equality
- Two of the motive waves in a five-wave sequence tend toward **equality in time and magnitude**
- Generally true of the two **non-extended** waves
- If perfect equality is lacking → **.618 multiple** is the next likely relationship
- For waves larger than Intermediate degree, use **percentage** terms
- For Intermediate degree and less, **arithmetic** terms are acceptable

### 5.5 Channeling
- Parallel trend channels mark upper and lower boundaries of impulse waves
- **Initial channel** (after wave 3 ends): Connect points 1 and 3; draw parallel through point 2 → estimates wave 4 boundary
- **Final channel** (after wave 4 ends): Connect ends of waves 2 and 4; draw parallel through peak of wave 3 → estimates wave 5 boundary
- If wave 3 is abnormally strong/vertical → use parallel through peak of wave 1 instead
- **Throw-over**: 5th wave penetrates upper channel line
  - Declining volume → 5th will meet or fall short of upper line
  - Heavy volume → possible penetration (throw-over)
  - Confirmed by immediate reversal back below the line
  - Preceded by "throw-under" by wave 4 or wave 2 of 5
- Use BOTH arithmetic and semilog scale

### 5.6 Volume
- Volume expands and contracts with speed of price change
- Wave 3 typically has the greatest volume
- Wave 5 volume usually less than wave 3 (below Primary degree)
- If wave 5 volume >= wave 3 volume → **fifth wave extension is in force**
- Low volume at end of correction = drying up of selling pressure → turning point
- Ending diagonals: diminishing volume, spike at throw-over point
- At Primary degree and greater, volume tends to be higher in advancing 5th wave

### 5.7 The "Right Look"
- Overall appearance must conform to expected proportions
- Cannot force a five-wave count into a three-wave count (or vice versa) just because internal count is ambiguous
- Disproportionate wave relationships or misshapen patterns should be rejected
- **Form analysis always takes precedence over ratio analysis**

### 5.8 Orthodox Tops and Bottoms
- Sometimes a pattern's end differs from the associated price extreme
- End of the pattern = "orthodox" top or bottom
- Examples: In a truncation, end of wave 5 is orthodox top despite wave 3's higher price
- In expanded flat, start of wave A is orthodox top despite wave B's higher high
- Wave lengths and projections should measure from orthodox ending points

## 6. WAVE PERSONALITY (behavioral expectations)

| Wave | Characteristics |
|---|---|
| **1** | ~50% are part of "basing" process, heavily corrected by W2; other 50% rise from large bases, dynamic, only moderately retraced |
| **2** | Often retraces most of W1; fear returns; low volume/volatility = drying up of selling pressure; produces Dow Theory "buy spots" |
| **3** | STRONGEST, broadest; greatest volume and price movement; usually extended; breakouts, "continuation" gaps, volume expansions; virtually all stocks participate |
| **4** | Predictable depth (via depth guideline) and form (alternation with W2); sideways trending; builds base for final W5; lagging stocks begin declining |
| **5** | Less dynamic than W3; slower max speed of price change; narrowing breadth; optimism extremely high; can still extend |
| **A** | Often mistaken for pullback in bull market; 5-wave A → zigzag for B; 3-wave A → flat/triangle for B |
| **B** | "Phonies" / bull traps / sucker plays; narrow participation; technically weak; doomed to complete retracement by wave C |
| **C** | Devastating; third-wave properties (strong, persistent, broad); illusions from A and B evaporate; fear takes over |
| **D** | Increased volume (non-expanding triangles); hybrid: part corrective but with first-wave characteristics; as phony as B waves |
| **E** | Appears as new trend kickoff; accompanied by strongly supportive news; often stages false breakdown through triangle boundary; emotional psychology like 5th waves |

## 7. FIBONACCI RATIO RELATIONSHIPS

### Key Ratios
| Ratio | Inverse | Primary Use |
|---|---|---|
| 0.236 | 4.236 | Shallow retracement; W4 of strong impulse |
| 0.382 | 2.618 | Common W4 retracement of W3 |
| 0.500 | 2.000 | Common retracement |
| 0.618 | 1.618 | **Golden ratio**; most common retracement; W2 of impulse |
| 0.786 | 1.272 | Deep W2 retracement; sqrt(0.618) |
| 1.000 | 1.000 | Equality (W5=W1; C=A in zigzag) |
| 1.236 | — | Wave B in expanded flat |
| 1.382 | — | Wave B in expanded flat; C extends beyond A |
| 1.618 | 0.618 | **Golden ratio**; most common extension; W3=1.618×W1 |
| 2.618 | 0.382 | Strong extensions; W3 in powerful moves |
| 4.236 | 0.236 | Extreme extensions |

### 7.1 Retracements
- **Wave 2**: typically 50%, 61.8%, or 78.6% of Wave 1
  - Sharp corrections (zigzags) tend to retrace **61.8% or 50%**
  - Especially when W2 of impulse, B of larger zigzag, or X in multiple zigzag
- **Wave 4**: typically 23.6%, 38.2%, or 50% of Wave 3
  - Sideways corrections (flats/triangles) tend to retrace **38.2%**
- **Wave B in zigzag**: 38.2% to 78.6% of Wave A
- **Wave B in flat**: ~100% of Wave A (regular), 100-138.2% (expanded)

### 7.2 Motive Wave Multiples
- **Wave 3**: often 1.618 × Wave 1 (or 2.618 × Wave 1 if strongly extended)
- **Wave 5 (non-extended)**: often equals Wave 1, or 0.618 × Wave 1
- **Wave 5 (extended)**: 1.618 × (Wave 1 through Wave 3 combined distance)
- **Wave 5**: sometimes related to W1-W3 net distance by 0.382 or 0.618
- **When W1 is extended**: Wave 2 often divides entire impulse into Golden Section
- **Wave 4 divides price range**: last portion = 0.382 (W5 not extended) or 0.618 (W5 extended)
  - The exact point within W4 that effects the subdivision varies (start, end, or extreme countertrend point)
  - This provides 2-3 closely-clustered targets for end of W5

### 7.3 Corrective Wave Multiples
- **Zigzag**: C usually = A (or 1.618×A, or 0.618×A)
- **Double zigzag**: second zigzag related to first by same ratios
- **Regular flat**: A ≈ B ≈ C
- **Expanded flat**: C often = 1.618×A; or C extends beyond A by 0.618×A; B = 1.236 or 1.382×A
- **Rare expanded flat**: C = 2.618×A
- **Triangle alternates**: at least two alternate waves related by 0.618 (e=0.618c, c=0.618a, d=0.618b)
- **Expanding triangle**: multiple is 1.618 instead of 0.618
- **In rare cases**: adjacent triangle waves related by these ratios
- **Double/triple corrections**: net travel of one pattern related to another by equality or 0.618
- **Wave 4 vs Wave 2**: often equality or Fibonacci relationship in price range

### 7.4 Multiple Wave Relationships (clustering)
- Future levels creating **MULTIPLE Fibonacci relationships** have greater likelihood of marking a turn
- When .618 retracement of larger wave, 1.618 multiple of subwave, AND 1.00 equality of internal wave all point to same price → **very strong target**
- All degrees of trend operate simultaneously → market is always full of Fibonacci relationships at various degrees

### 7.5 Time Relationships
- Fibonacci time spans between turning points (8, 13, 21, 34, 55 periods)
- Time periods produce tops-to-bottoms, tops-to-tops, bottoms-to-bottoms, bottoms-to-tops
- Bear markets tend to run **61.8% of the time** required for bull markets
- Most useful when they coincide with price targets and wave counts
- "Permutations tend to become infinite" — use as supporting evidence, not primary

## 8. ALGORITHMIC IMPLEMENTATION GUIDE

### 8.1 Pattern Decision Tree

#### Step 1: Determine Expected Mode
- If parent wave is actionary (1, 3, 5, A, C) → expect motive (5-wave)
- If parent wave is reactionary (2, 4, B, D, X) → expect corrective (3-wave)

#### Step 2: Impulse Validation Checklist
- [ ] 5 alternating pivots identified
- [ ] Wave 2 does not retrace > 100% of Wave 1
- [ ] Wave 3 end exceeds Wave 1 end (in trend direction)
- [ ] Wave 3 is not shortest of 1, 3, 5 (in price movement)
- [ ] Wave 4 does not overlap Wave 1 territory
- [ ] If Wave 4 overlaps Wave 1 → check for diagonal triangle
- [ ] Alternation: Wave 2 and Wave 4 differ in form/depth/complexity

#### Step 3: Correction Classification
Count significant pivots and check internal structure:
- 2 pivots → simple ABC: check subwave structure for zigzag (5-3-5) vs flat (3-3-5)
- 4 pivots → double three (W-X-Y) or triangle start
- 5+ pivots → triangle (a-b-c-d-e) or triple three (W-X-Y-X-Z)

#### Step 4: Diagonal Triangle Detection
If impulse rules are violated but 5-wave structure exists:
- Check if W4 overlaps W1 AND boundary lines converge
- If in wave 5 or C position → ending diagonal (3-3-3-3-3)
- If in wave 1 or A position → leading diagonal (5-3-5-3-5)

#### Step 5: Projection Selection
Based on wave in progress, apply most probable Fibonacci targets (section 7).

### 8.2 Confidence Scoring Model
Rate each wave interpretation based on:
1. **Rule compliance** (mandatory pass/fail): R1-R5
2. **Guideline satisfaction** (weighted scoring):
   - Alternation between W2/W4 (+weight)
   - Fibonacci ratio relationships (+weight per relationship found)
   - Channeling adherence (+weight)
   - Wave equality (+weight)
   - Wave personality match (+weight: W3 strongest, etc.)
   - Volume pattern match (+weight)
   - Depth of correction matches expected level (+weight)
3. **Multiple relationship clustering** at projected targets (+high weight)
4. **The "right look"** — overall proportionality

### 8.3 Alternate Count Management
- Always maintain at least 2 valid wave interpretations
- Preferred = satisfies most guidelines; alternate = next best
- If market violates preferred count → alternate becomes new preferred
- Both counts must satisfy ALL inviolable rules
- Re-evaluate when subsequent waves clarify the picture

### 8.4 Deductive Reasoning Approach
- Elimination process: "When you have eliminated the impossible, whatever remains must be the truth"
- Know what Elliott rules will NOT allow
- Deduce most likely course from what remains
- Apply extensions, alternation, overlapping, channeling, volume, ratios
- Don't allow emotional involvement to accept misshapen patterns

### 8.5 Practical Wave Counting Tips
- Use semilog scale for large degree waves; arithmetic for small degree
- Use both scales always for channeling
- When apparent W3 is too short: relabel as extended wave (3) in the making
- 5-wave A in correction → expect zigzag (sharp, deep correction)
- 3-wave A in correction → expect flat or triangle (sideways correction)
- If B wave shows 5-wave structure (not 3) → likely W1 of new impulse, not B
- "If something seems wrong with this market" → probably a B wave

## 9. SUMMARY OF ALL PATTERNS

| Pattern | Structure | Position | Key Features |
|---|---|---|---|
| Impulse | 5-3-5-3-5 | 1,3,5,A,C | No overlap; W3 never shortest |
| Extension | elongated subwave | within impulse | Only one of 1,3,5; W3 most common |
| Truncation | W5 < W3 end | wave 5 | After strong W3; verify 5 subwaves |
| Ending Diagonal | 3-3-3-3-3 | 5 or C | Overlap allowed; wedge; exhaustion |
| Leading Diagonal | 5-3-5-3-5 | 1 or A | Overlap allowed; wedge; continuation |
| Zigzag | 5-3-5 | 2,A,W,Y,Z | Sharp; B noticeably < A start |
| Double Zigzag | ZZ-X-ZZ | 2,A,W,Y | Deeper correction needed |
| Flat (Regular) | 3-3-5 | 4,B | B ≈ A start; C slightly past A end |
| Flat (Expanded) | 3-3-5 | 4,B | B > A start; C well past A end |
| Flat (Running) | 3-3-5 | 4,B | B > A start; C short of A end; rare |
| Triangle | 3-3-3-3-3 | 4,B,final X | Overlapping; converging/expanding lines |
| Double Three | W-X-Y | corrective | 1 zigzag max, triangle only as final |
| Triple Three | W-X-Y-X-Z | corrective | Same rules; extends duration |
